Some Web API

Web Audio API

Support

if (!window.AudioContext) {
	// API not supported :(
} else {
	// API supported! Let's start the fun :)
}

Sources:

  • audioBufferSourceNode
  • mediaElementAudioSourceNode
  • mediaStreamAudioSourceNode

Destinations:

  • context.destination
  • mediaStreamAudioDestinationNode

Schema

source1.connect(node1); 
source2.connect(node3); 
node1.connect(node4); 
node1.connect(node2); 
node2.connect(destination); 
node3.connect(node1); 
node4.connect(destination); 
node4.connect(node3);

Modules:

  • Gain
  • Delay
  • Convolution
  • Filter
  • Analyser
  • Generator
  • ...

How to use:

const context = new AudioContext();
const source = context.createBufferSource();
source.buffer = getBuffer();
const destination = context.destination;

const gainNode = context.createGain();
const delayNode = context.createDelay();
gainNode.gain.value = 0.5;
delayNode.delayTime.value = 0.75;

source.connect(gainNode);
gainNode.connect(delayNode);
delayNode.connect(gainNode);
gainNode.connect(destination);
source.start();

Speech Recognition API

Support

if (!window.SpeechRecognition) {
	// API not supported :(
} else {
	// API supported! Let's start the fun :)
}

SpeechRecognition:

  • continuous
  • lang
  • interimResults
  • start()
  • stop()
  • onstart()
  • onresult()
  • onerror()
  • onend()

How to use:

const recognizer = new window.SpeechRecognition();
recognizer.continuous = true;
recognizer.interimResults = true;
recognizer.onresult = (event) => {
	let textContent = '';
	Array.from(event.results).forEach(res => {
		if (res.isFinal) {
			textContent += `${res[0].transcript}\n`;
		}
	});
};
recognizer.start();

Web Notification API

Support

if ('Notification' in window) {
	// API supported
} else {
	// API not supported
}

Permission:

Notification.requestPermission().then((permission) => { ... });
  • default
  • granted
  • denied

Notification:

const notification = new Notification(title, options);

Properties:

  • title
  • options
    • body
    • dir
    • lang
    • tag
    • icon

How to use:

if (!("Notification" in window)) {
	alert("This browser does not support desktop notification");
} else if (Notification.permission === "granted") {
	const notification = new Notification("Hi there!");
} else if (Notification.permission !== 'denied') {
	Notification.requestPermission((permission) => {
		if (permission === "granted") {
			const notification = new Notification("Hi there!");
		}
	});
}

Battery Status API

Support

if (window.navigator.getBattery()) {
	// Grab the battery's information!
} else {
	// Not supported
}

BatteryManager:

  • charging
  • chargingTime
  • dischargingTime
  • level

Events:

  • onchargingchange()
  • onchargingtimechange()
  • ondischargingtimechange()
  • onlevelchange()

How to use:

let batteryIsCharging = false;

navigator.getBattery().then((battery) => {
	batteryIsCharging = battery.charging;

	battery.addEventListener('chargingchange',() => {
		batteryIsCharging = battery.charging;
	});
});

Device Orientation API

Support

Detecting support:

if (window.DeviceOrientationEvent) {
	// We can listen for change in the device's orientation...
}

if (window.DeviceMotionEvent) {
	// We can listen for change in the device's orientation...
}

if (!('oncompassneedscalibration' in window)) {
	// Event supported
}

Orientation Event:

  • deviceorientation
    • alpha
    • beta
    • gamma
  • deviceorientation
    • acceleration
    • accelerationIncludingGravity
    • rotationRate
    • interval
  • compassneedscalibration

How to use:

window.addEventListener('deviceorientation', 
	(e) => {
		console.log('a = '+e.alpha); 
		console. log('b = '+e.beta);
		console. log('g = '+e.gamma);
});
window.addEventListener('devicemotion', 
	(e) => {
		console.log('x = '+Math.round(e.acceleration.x)); 
		console. log('y = '+Math.round(e.acceleration.y));
		console. log('z = '+Math.round(e.acceleration.z));
});
window.addEventListener('compassneedscalibration',
	(e) => { alert('Compass needs calibrating!'); }
);
